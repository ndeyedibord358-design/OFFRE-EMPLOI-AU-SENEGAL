# 🇸🇳 Analyse des Offres d'Emploi au Sénégal

## 📌 Description

Ce projet est une **analyse de données** sur les offres d'emploi disponibles au Sénégal. Les données ont été collectées par **web scraping** sur deux plateformes d'emploi :

- [AgroAfricaOnline](https://www.agroafricaonline.com)
- [SenJob](https://www.senjob.com)

L'objectif est d'aider les chercheurs d'emploi à mieux comprendre le marché du travail sénégalais en visualisant les offres par **région**, **secteur**, **type de contrat**, **compétences** et **catégories**.

---

## 🎯 Objectifs

- Collecter les offres d'emploi disponibles au Sénégal
- Nettoyer et structurer les données
- Analyser les tendances du marché de l'emploi
- Visualiser les résultats de manière claire et interactive

---

## 🔍 Fonctionnalités

- 📍 **Filtrage par région** — Voir les offres disponibles par zone géographique
- 🏭 **Filtrage par secteur** — Identifier les secteurs qui recrutent le plus
- 📄 **Filtrage par type de contrat** — CDI, CDD, Stage, Freelance...
- 🏆 **Top compétences** — Les compétences les plus demandées par les employeurs
- 📂 **Catégories d'emploi** — Répartition des offres par catégorie professionnelle

## 🚀 Comment utiliser ce projet

1. Clone le dépôt :
```bash
git clone https://github.com/ndeyedibord358-design/OFFRE-EMPLOI-AU-SENEGAL.git
```

2. Ouvre le notebook sur **Google Colab** ou **Jupyter Notebook**

3. Installe les dépendances nécessaires :
```bash
pip install pandas requests beautifulsoup4 matplotlib seaborn
```

4. Lance le notebook et explore les analyses !
